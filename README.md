# 🌐 Personal Portfolio - Guilherme Marin

A modern, responsive personal portfolio website showcasing my journey as an Electronic Engineering and Computer Science student. It highlights my professional experience in industrial automation, software development, and artificial intelligence, as well as my academic background in Brazil and Germany.

## 🔗 Live Demo
https://guimarin4.github.io/LandingPage-GuilhermeMarin/

## ✨ Features

*   **Clean & Modern UI:** Built with custom CSS, utilizing `anim-fade-in` and `anim-fade-up` classes for smooth scroll animations.
*   **Typography:** Uses modern Google Fonts (*Syne* for headings and *DM Mono* for technical details).
*   **Experience Timeline:** Details my roles across different countries, from product validation in Brazil to hospitality in Munich.
*   **Skills Dashboard:** Visual representation of my technical stack divided into three core areas:
    *   🔌 Embedded & Automation (PLC, EPLAN, Microcontrollers)
    *   💻 Software & Web (Python, React, C++, Docker)
    *   🧠 AI & Data (PyTorch, Pandas, CPLEX)
*   **Multilingual Support:** Showcases proficiency in Portuguese, English, Spanish, and German.
